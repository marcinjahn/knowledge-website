# CSS Pixel

One of the units that we can use in CSS is a px. The name is confiusing, because the CSS px does not represent the display pixel. Instead, it is a Reference Pixel (opens new window). The general rule is that 96 px (CSS px) = 1 inch. This is physically true on hi-dpi displays, and less true on low-dpi ones. W3C initially required all CSS engines to render absolute sizes in a correct way, according to physical dimensions. However, manhy implementations did not follow that and W3C eventually dropped that requirement (source (opens new window)).

It is important, because it means that our current webpages that have sizes defined in px will continue to work in the future as higher resolution screens are developed.
